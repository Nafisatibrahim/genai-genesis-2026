# frontendedit — Teammate Editable Copy

This folder is a copy of `frontendapp/` for teammates to freely edit and experiment with.

## Rules

- **Edit freely here** — this is your sandbox.
- **Do NOT edit `frontendapp/`** — that is the locked, running production version on Replit.
- The live app is served from `frontendapp/`, not this folder.

## Getting started

```bash
cd frontendedit
npm install
npm run dev
```

The backend API is at `https://flex-flow.up.railway.app` — create a `.env` file:

```
VITE_API_URL=https://flex-flow.up.railway.app
```

## When your changes are ready

Tell the project owner to review and merge your changes from `frontendedit/` into `frontendapp/`.
